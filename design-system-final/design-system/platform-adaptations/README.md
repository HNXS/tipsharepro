## Web (Current Focus)

TipSharePro is a web-first application targeting:

- Modern browsers (Chrome, Firefox, Safari, Edge)
- Responsive design (mobile to desktop)
- Print optimization for distribution reports
- Touch-friendly interactions

See [web.md](./web.md) for complete guidelines.

---

## Future: Mobile Apps

If native mobile apps are developed:

- Adapt components to platform conventions
- Use native navigation patterns
- Support system dark mode
- Implement haptic feedback
- Follow iOS HIG / Material Design as appropriate
