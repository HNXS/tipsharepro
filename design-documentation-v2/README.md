## Overview

This design system is built to match the bold, professional aesthetic of the current TipSharePro.com website. Per Tom's feedback: "Don't get too enamored with pastel accents... throw in some color or I'll fade off to sleep. Take a look at my Logo. Not bashful."

## Design Philosophy

**POWERFUL * FAIR * TRANSPARENT**

The design embodies these three brand pillars:
- **Powerful:** Bold colors, confident typography, strong visual hierarchy
- **Fair:** Clean layouts, balanced spacing, accessible design
- **Transparent:** Clear information architecture, honest visual feedback

## Key Design Principles

1. **Bold Over Subtle** - Use strong, saturated colors from the logo palette
2. **Professional Confidence** - Restaurant industry professionals expect serious tools
3. **Warm & Approachable** - Kitchen/restaurant imagery creates connection
4. **Clean But Not Sterile** - Strategic use of color prevents "boring spreadsheet" feel
